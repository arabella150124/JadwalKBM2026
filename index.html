Sambungkan Cloud Firestore

Masukkan konfigurasi aplikasi Firebase Anda untuk mensinkronisasikan data ke database Cloud.

Bagaimana Cara Mendapatkan Konfigurasi Ini?

  1. Buka Firebase Console, buat proyek baru atau gunakan yang sudah ada.
  2. Aktifkan Cloud Firestore Database di menu Build.
  3. Pergi ke Project Settings (Ikon roda gigi) -> Pilih platform Web App (Ikon </>), daftarkan aplikasi Anda.
  4. Salin variabel objek firebaseConfig dan masukkan kolom-kolomnya di bawah ini.
